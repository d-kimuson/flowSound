$

## Usage

- **flowSound/exe/dist/Prototype.app**
- **flowSound/exe/dist/connect**
- **flowSound/src/connect.py**

からそれぞれ実行できる.

        ├── fin.ui
        ├── guide.ui

        ├── result.ui
